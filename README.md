# BHSFMathWeb
## Frontend Install
In `frontend/` dir
1. main project
   ```
   npm run dev
   ```
2. antd
    ```
    npm install antd --save
    ```
3. link
   ```
   npx @next/codemod new-link .
   ```
4. zustand
   ```
   npm install zustand
   ```
## Backend Start
```
python manage.py runserver
```
